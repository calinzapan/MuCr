MuCr (Music Collector) - Progresul proiectului

Descriere

MuCr (Music Collector) este o platforma online de gestionare a colectiilor muzicale destinata iubitorilor de muzica.Acestia se pot organiza in grupuri pastrandu-si colectiile intr-un bloc comun. Aceștia au capacitatea de a inregistra, prin completarea unor formulare specifice, discuri de vinyl, benzi magnetice, bilete pentru spectacole, precum și melodii.

Introducere

Echipa este formată din :Mîrza Constantin-Bogdan , Rîpan Ilie-Vladimir și Zapan Călin-George. Vom prezenta in continuare detalii despre acest proiect.

După cum a fost mentionat in descriere, s-a pus la dispozitia utilizatorilor o aplicatie web pentru gestionarea colectiilor muzicale.

Detalii despre modul de lucru :

  • Dezvoltarea front-end-ului (HTML + CSS + JavaScript).
  • Realizarea arhitecturii.
  • Intelegerea modului de lucru cu functii callback.
  • Implementarea design patternului MVC (Model View Controller).
  • Integrarea modulelor obtinute pe baza arhitecturii construite.
  • Testarea fiecarui modul.

In final, s-a reușit realizarea unui proiect functional.

Structura

In continuare, vom prezenta tehnologiile folosite :

HTML

Cu ajutorul HTML s-a reusit structurarea paginilor web astfel incat sa putem expune intr-o structura bine definita informatia primita de la server.

CSS

CSS-ul a fost folosit pentru a infrumuseta paginile si pentru a obtine o pagina responsive.

Baze de Date

Ca si metoda de stocare a datelor, a fost folosita o baza de date Oracle accesata prin intermediului serviciului RDS furnizat de Amazon. Pentru a comunica cu baza de date s-a folosit softul DataGrip realizat de JetBrains.

JavaScript-Front-End

Pe front end s-a folosit JavaScript pentru a realiza requesturi Ajax la server si pentru a modifica stilul paginilor in functie de evenimentele generate de utilizator.

JavaScript-Back-End

Pe back end s-a folosit JavaScript alaturi de modulele NodeJS pentru a realiza serverul web.
